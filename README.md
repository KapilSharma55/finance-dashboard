# Finance Dashboard Backend

## Overview
This is a Spring Boot backend application for managing financial records and generating dashboard summaries.

## Features
- User and Role Management
- Financial Records CRUD
- Dashboard Summary (Income, Expense, Balance)
- Role-based Access Control (Spring Security)

## Tech Stack
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL

## Run Instructions
1. Configure MySQL in application.properties
2. Run the project using:
   mvn spring-boot:run
